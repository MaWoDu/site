Practical Server-less Architecture using AWS

Created by Mashooq Badar

Building your Web Application is complicated enough, you don't want to build the infra as well. Cloud based infra such as AWS gives you a tried and tested deployemnt platform so you can concentrate on the business. This course will give you hands-on experience of how to build fully scalable, reliable and secure web apps using the AWS server-less architecture.

More courses Contact

Course Duration

2 days

Why Attend?

  • Learn how build web apps using AWS Server-less Architecture.
  • Learn how to adhere to the AWS Well Architected Framework.
  • Learn the significance of the various AWS products within this type of architecture.

Content

  • Build scalable REST APIs using API Gateway and AWS Lambda.
  • Build scalable persistence using RDS with automated backups.
  • Secure using IAM, KMS, AWS WAF, Json Web Token, and SSL.
  • Monitor using Cloud Trail and Cloud Watch.
  • Serve static content using S3.
  • Deploy using AWS CLI.

Audience

Developers and Teams interested in building secure, reliable and scalable web applications using the Amazon Server-less architecture.

Prerequisites

  • Bring your own laptop.
  • Linux VM installed if working from a windows laptop.
  • Be proficient with a JVM language (Java, Clojure, Scala or Groovy), Node.js or Python.
  • Have your favourite development environment installed and ready.
